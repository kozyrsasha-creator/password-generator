About GetPassnow

A tiny project with a simple idea: create strong passwords locally, without trackers, accounts, or servers in the middle.

Why this exists

Most people reuse weak passwords across multiple accounts. We wanted something small and fast that helps anyone make strong, unique credentials right in the browser. No sign-ups, no telemetry — just a reliable generator powered by modern Web Crypto APIs.

What we focus on

Privacy by default

Passwords are generated on your device and never transmitted. We avoid tracking cookies and third-party analytics by default.

Clear controls

Length slider, character sets, and options like no repeats or excluding ambiguous characters — all to match website rules and usability.

Actionable feedback

The strength meter, entropy estimate, and rough time-to-crack offer context so you can pick safer defaults for important accounts.

Multi-language

We’re adding localized pages so more people can use the tool comfortably and search engines can serve the right language.

Security basics we recommend

  • Use a different password for every account.
  • Aim for 16+ characters; go longer for sensitive services.
  • Enable two-factor authentication (2FA) wherever possible.
  • Store credentials in a reputable password manager.

Roadmap

  • Optional passphrase mode (random word sequences)
  • Theme toggle stored locally
  • Accessibility & keyboard navigation refinements

If GetPassnow helps you, share it with a friend or colleague. Strong, unique passwords make the web safer for everyone.
